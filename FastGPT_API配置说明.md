# FastGPT API 密钥配置说明

## ❌ 当前问题

您遇到的错误：
```
"Key is error. You need to use the app key rather than the account key."
```

**原因**：您使用的是账户级别的通用密钥，但需要使用应用专用的API密钥。

## 🔑 两种密钥类型

### 1. 账户密钥（Account Key）
- ❌ **不能用于**：调用应用对话接口
- ✅ **可以用于**：管理账户、创建应用等
- 📍 **获取位置**：账户设置 → API密钥

### 2. 应用密钥（App Key）
- ✅ **可以用于**：调用v1对话接口 `/api/v1/chat/completions`
- ✅ **推荐使用**：本项目需要的就是这种密钥
- 📍 **获取位置**：我的应用 → 选择具体应用 → API密钥

## 🎯 正确获取应用密钥的步骤

### 1. 登录FastGPT平台
访问：https://fastgpt.in

### 2. 进入应用管理
点击 **"我的应用"** 或 **"应用管理"**

### 3. 选择或创建应用
- 如果已有应用：点击进入具体应用
- 如果没有应用：先创建一个新应用

### 4. 找到API密钥设置
在应用页面中找到：
- **"API密钥"** 或 
- **"API Key"** 或 
- **"应用设置"** → **"API密钥"**

### 5. 复制应用专用密钥
- 复制显示的应用API密钥
- 确保是该应用专用的密钥

## 🔧 配置密钥

### 方法一：控制台配置
在浏览器控制台执行：
```javascript
setFastGPTKey('fastgpt-应用专用密钥')
```

### 方法二：查看当前配置
```javascript
showConfig()
```

## ✅ 验证密钥

正确的应用密钥特征：
- ✅ 以 `fastgpt-` 开头
- ✅ 从具体应用页面获取
- ✅ 可以调用该应用的对话接口

## 📋 常见问题

**Q: 账户密钥为什么不能用？**
A: FastGPT的v1对话接口专门设计为应用级别的API，需要应用专用密钥来确保安全性和权限控制。

**Q: 如何区分账户密钥和应用密钥？**
A: 获取位置不同，账户密钥在账户设置中，应用密钥在具体应用页面中。

**Q: 没有应用密钥能用系统吗？**
A: 可以！系统有高质量的本地模板，功能完全可用。

---

**总结**：请获取应用专用的API密钥，而不是账户通用密钥。系统在任何情况下都可以正常使用！
